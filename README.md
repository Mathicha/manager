# Modules Manager

Load / Unload [tera-proxy](https://meishuu.github.io/tera-proxy) modules with chat [command](https://github.com/pinkipi/command).

# Commands

**`\8 modules`**

* Show help and current loaded modules.

**`\8 load`**

* load a module.

**`\8 unload`**

* Unload a module.

**`\8 reload`**

* Reload a module.
